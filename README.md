# Detección de Objetos con YOLOv8 en Google Colab

## Descripción
Este proyecto implementa un sistema de detección automática de objetos en imágenes utilizando el modelo YOLOv8 y técnicas de visión por computadora.

La ejecución se realiza completamente en Google Colab, aprovechando el uso de GPU para acelerar la inferencia del modelo.

## Tecnologías utilizadas
- Python 3
- Google Colab
- YOLOv8 (Ultralytics)
- OpenCV
- Deep Learning

## Funcionalidades
- Detección automática de múltiples objetos
- Visualización de bounding boxes
- Clasificación de objetos
- Uso de modelos preentrenados
- Ejecución en GPU

## Ejecución

1. Abrir el archivo `.ipynb` en Google Colab
2. Ejecutar cada celda en orden
3. Subir una imagen para detección

## Dataset

Modelo entrenado con el conjunto de datos COCO.
